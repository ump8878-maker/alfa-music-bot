# Деплой на Railway

Чтобы работала **новая** версия бота (скан чата, рейтинг, различие личка/группа):

1. **Репозиторий** — должен быть подключён тот же GitHub-репозиторий, где лежит этот код (main.py, handlers/, без папки app/).

2. **Ветка** — `main`.

3. **Команда запуска** — в настройках сервиса оставь как в Procfile:
   ```
   worker: python run.py
   ```
   Не должно быть `python app/bot.py` или `python run.py` с другим путём.

4. **Переменные** — `BOT_TOKEN` (обязательно), при необходимости `DATABASE_URL`.

5. **Пересборка** — после пуша в main Railway обычно деплоит сам. Если видишь старый бот:
   - В Railway: Deploy → Redeploy (или Clear build cache + Redeploy).
   - Убедись, что в логах при старте есть строка с `(entry: main.py)`.
   - В личке напиши боту `/start` — внизу должно быть *«v2 · скан и рейтинг чатов»*. Если этой строки нет, крутится старая версия.

6. **Один инстанс** — не запускай того же бота локально с тем же токеном, иначе будет TelegramConflictError.
